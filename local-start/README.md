# Snabb Local Start

Run `start-snabb.bat` from this folder to launch the project locally.

It starts:

- Backend FastAPI server from `..\backend` on `http://localhost:8001`
- Web Next.js app from `..\web` on `http://localhost:3000`
- Mobile Expo app from `..\mobile` on port `8081`

The launcher checks that the backend virtual environment and Node dependencies exist before opening the service windows.
